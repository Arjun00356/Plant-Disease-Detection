Plant Leaf Disease Detection using Deep Learning

A deep learning based system that classifies plant species and detects whether a leaf is healthy or diseased using image classification.

üß† Model Architecture

Transfer learning with MobileNetV2 is used for efficient image classification.

  1. Image preprocessing (resize 224√ó224, normalization)
  2. Feature extraction using pretrained MobileNetV2
  3. Global Average Pooling layer

  5. Dropout for overfitting prevention
  6. Softmax output for multiclass classification
